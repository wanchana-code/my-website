Past Tense

The past tense is used to describe actions or events that have been completed in the past.

Example: "She walked to the store yesterday."

Present Tense

The present tense is used to describe actions or events that are currently happening or are always true.

Example: "She walks to the store every day."

Future Tense

The future tense is used to describe actions or events that will happen in the future.

Example: "She will walk to the store tomorrow."
